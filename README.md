# game-of-life
Game Of Life made using raylib/raylib-cpp and imgui.
![image](https://github.com/user-attachments/assets/7dc89e26-b668-4bd8-95a0-02c446be63ac)

## Build steps
```
mkdir .build && cd .build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
```

## Run game
```
cd .build
./core/game-of-life
```
